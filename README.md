# Universally — WordPress Plugin

Translate your WordPress site into 110+ languages with AI. Multilingual plugin with SEO-friendly URLs, hreflang tags, language switcher, and glossary rules.

**[Install from WordPress.org →](https://wordpress.org/plugins/universally-language-translation-multilingual-tool/)**

Translations are powered by [Universally](https://universally.com/?utm_source=github&utm_medium=link&utm_campaign=plugin_repo_readme) — bring your own API key from [universally.com/pricing](https://universally.com/pricing/?utm_source=github&utm_medium=link&utm_campaign=plugin_repo_readme).
